Where's Waldo International - Facebook Hackathon, Nov. 2013

Three team members and I programmed a game of Where's Waldo. Using the Google Maps API, we give Waldo a random location around the world. The player first receives a Javascript message with a clue for the player to guess which continent Waldo is on. Once the right continent is identified, the player is given a Google Maps marker and can move it around the continent until Waldo is found. There is a hot-cold bar at the top of the map to indicate how close or far away the player's current pointer location is from Waldo's actual location. Once Waldo is found, the player is given a score out of 100 based on the time taken to find Waldo. The player can share the game via different social media sites.

Languages Used: HTML, CSS, Javascript, Google Maps API, Facebook API, Twitter API, Google Plus API, Tumblr API
